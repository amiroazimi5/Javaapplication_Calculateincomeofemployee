راهنمای فارسی :
برنامه در ابتدا ، تعداد کارکنان را میپرسد و اطلاعات مورد نیاز برای محاسبه را از کاربر میگیرد. سپس اطلاعات را به صورت یک لیست به کاربر نمایش داده و در صورت که هر کدام از کارکنان دارای بیمه یا حق اولاد باشند ، هزینه ی آنها در کد محاسبه خواهد شد و حقوقی که در لیست برای آن فرد ، تغییر خواهد کرد.

نکته : برنامه فقط با نام های انگلیسی کار خواهد کرد.

English Instruction :
The program first asks the user for the number of employees and takes the information needed for the calculation. Then, it displays the information to the user in the form of a list, and if any of the employees have insurance or child benefits, their costs will be calculated in the code and the salary in the list for that person will change.

note : app work Only with English Names Of employees!
